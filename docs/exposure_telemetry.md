# Exposure Telemetry

## Purpose

This file records exposure-related telemetry for the V13 LoopKit repo.

It is not a field note.
It is not a claim that a specific README change caused a specific traffic outcome.
It is a lightweight ledger for views, visitors, clones, cloners, and nearby exposure context.

## Recording Rules

* Record observed numbers.
* Do not infer human adoption from clone counts alone.
* Do not claim causality from README changes without stronger evidence.
* Note nearby README, outreach, or exposure events.
* Use the log to support later V13 exposure-learning decisions.
