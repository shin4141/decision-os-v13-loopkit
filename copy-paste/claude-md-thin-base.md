# Thin CLAUDE.md / AGENTS.md Base

Use this as a small always-loaded base for AI coding agents.

Keep this file short.

Do not put every project note, preference, or past mistake here.

This file is only for rules that should apply across almost every task.

## Always-loaded rules

- Keep this file short.
- Treat this file as the operating manual, not the full project memory.
- Do not treat every saved note as an always-on rule.
- For task-specific instructions, read the relevant file before acting.
- If unsure whether a rule applies, ask before continuing.
- Do not invent features, files, or implementation details.
- Do not start the next task automatically after finishing the current one.

## Before acting

Check:

- What is the current task?
- Which files are relevant?
- Which instructions are task-specific?
- What should not be touched?
- Is there a clear stop condition?

If the task, source of truth, or stop condition is unclear, ask before continuing.

## Before handing off

Report:

- What changed
- What was verified
- What remains unverified
- Source of truth
- Off-limits scope
- Next safe action
- What the next AI/human should not assume

Do not start the next action automatically.

## Rule of thumb

Always-loaded instructions should stay small.

Project memory, field notes, mistakes, examples, and task-specific guidance should live in separate files and be read only when relevant.
