# Contributing

Thank you for your interest in V13 LoopKit.

This repository is a prototype documentation and operating-protocol kit for AI-agent work after completion.

## Good contributions

Good contributions include:

- README clarity improvements
- examples that make first use easier
- field notes from real AI-agent tasks
- documentation fixes
- issue reports about confusing gates or unclear next-loop behavior

## Not currently in scope

Please do not add:

- automation
- CLI/server/package setup
- Telegram or notification bots
- schema expansion without an exposed gap
- broad productization
- V13 v1.0 drafting

## Before proposing changes

Ask:

1. What exposed gap does this repair?
2. Does it improve first-use clarity, restartability, handoff safety, or next-loop control?
3. Does it preserve the current public / feature-growth CAP?
4. Is it a 0.01 repair rather than speculative expansion?

## Reporting format

When proposing changes, include:

- problem
- proposed repair
- expected effect
- risk
- whether a Decision Packet is required
